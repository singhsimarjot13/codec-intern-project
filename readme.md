# 🤖 Machine Learning Projects

This repository contains two machine learning projects implemented in Python using real-world datasets.

## 📈 Stock Price Prediction using LSTM

A deep learning-based stock forecasting system that predicts future stock prices using historical market data.

### Technologies

* Python
* TensorFlow / Keras
* Pandas
* NumPy
* Scikit-Learn
* Yahoo Finance (yfinance)

### Results

* MAE: 6.39
* RMSE: 8.02
* R² Score: 0.90+

---

## 🎬 Movie Recommendation System

A content-based recommendation engine that suggests similar movies using movie metadata.

### Technologies

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Cosine Similarity
* KaggleHub

### Dataset

* TMDB Movie Metadata Dataset
* 4,800+ movies

### Features

* Content-Based Filtering
* Similarity Scoring
* Top-N Recommendations
